# 2026-07-19

## What I learned

Set up this TIL repo: dated entries with tag front matter, a `til.sh` helper
to scaffold new entries, and a pre-commit hook that regenerates the README
index automatically so I never have to update it by hand.

## Notes / details

- `./til.sh new` creates and opens today's file.
- Committing regenerates `README.md` via the `.githooks/pre-commit` hook.
- Multiple commits/pushes per day are fine — the log is just append-only files.
